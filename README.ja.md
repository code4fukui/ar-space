# ar-space

AR空間 - WebXRを使用した平面およびメッシュの検出。

## デモ
https://code4fukui.github.io/ar-space/

## 機能
- AR空間での平面およびメッシュの検出と表示
- 検出イベントをサーバーに送信してログを記録

## 要件
AR機能に対応したデバイスと、WebXR対応ブラウザ（Chrome、Edge、Firefoxなど）が必要です。

## 使い方
1. 対応ブラウザでデモページを開きます。
2. カメラおよびAR機能へのアクセスを許可します。
3. 検出された平面およびメッシュがシーン内に表示されます。

## データ / API
本プロジェクトでは、AR機能にWebXR APIを使用しています。
