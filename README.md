# ar-space

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

AR space - plane and mesh detection using WebXR.

## Demo
https://code4fukui.github.io/ar-space/

## Features
- Detect and display planes and meshes in AR
- Sends detection events to a server for logging

## Requirements
Requires a WebXR-enabled browser (e.g., Chrome, Edge, or Firefox) on a device with AR capabilities.

## Usage
1. Open the demo page in a compatible browser.
2. Grant permission to access the camera and AR features.
3. The detected planes and meshes will be displayed in the scene.

## Data / API
This project uses the WebXR API for AR functionality.
